Installation of Altprobe¶


Requirements¶

Altprobe was tested under Ubuntu version 16.04, 18.04 and Centos 7. Minimum Memory on a computer/virtual machine should be not less than 2GB, Hard disk not less 10GB.

Before starting this procedure, please install Cnode, after this you need to get info (file Broker.pem, project id) which are required for installation altprobe (see section Post-installation of cnode).


Installation¶

  • Login to Alertflex Collector and download installation files.

git clone https://github.com/alertflex/altprobe.git
cd ./altprobe

  • Fill in Altprobe specific parameters in the env.sh file, as an example use the file env_example.sh. If you are planning to install Altprobe on the Cnode compute use as example file env_local.sh.

  • Start installation.

chmod u+x install_ubuntu.sh
./install_ubuntu.sh

  • Copy Broker.pem file to directory /etc/altprobe/

  • Reboot system and after that check status of Altprobe

root@host:~# altprobe-status

  • Open Alertflex console, select “Alerts” and check if alerts from IDS are presented.

  • How to check an Altprobe errors:

cat /var/log/syslog | grep altprobe
