Open Source Security Automation and Monitoring Solution

Alertflex project is a free, open-source continuous security monitoring solution designed for use in Hybrid Clouds (on-premises and cloud-based IT infrastructure). The project integrates third-party software products into a unified solution based on the CTI EDR NTA SOAR technology stack and provides a single pane of view for more than 15 security products.

Alertflex requires fewer system resources compared with similar products and mostly integrates of free open-source software. It can significantly reduce costs for your cybersecurity monitoring.

Alerts filtering, prioritization and visualization

Detection intrusions, vulnerabilities and misconfigurations

Integrated analysis network, containers and hosts

Security operations automation and response
