Open source security orchestration solution

Alertflex project is a continuous security monitoring solution designed for use in Hybrid Clouds (on-premises and cloud-based IT infrastructure). By monitoring security events and reports from well-known open-source security applications, Alertflex gives companies end-to-end security visibility. The project adapts of third-party software products into a unified solution based on the CTI EDR SOAR technology stack and DevSecOps best practices.

Alerts filtering, prioritization and visualization

Detection intrusions, vulnerabilities and misconfigurations

Integrated analysis network, containers and hosts
